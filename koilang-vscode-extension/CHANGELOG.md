# Change Log

All notable changes to the KoiLang extension will be documented in this file.

## [0.1.0] - 2025-11-24

### Added
- Initial release of KoiLang language support
- Syntax highlighting for KoiLang commands starting with #
- Support for parameters, strings, numbers, and boolean values
- Comment highlighting (both line and block comments)
- Recognition of text content outside commands
- Support for complex data structures (lists and dictionaries)
- Basic language configuration for indentation and bracket matching
